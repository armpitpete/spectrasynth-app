# v0.77 — Source mix bus

Implements issue #125.

## Change

Adds one explicit source mix bus:

```text
sourceMixGain
```

Oscillator and Noise now route through it before the existing core path:

```text
oscillatorGain -> sourceMixGain -> toneFilter
noiseGain      -> sourceMixGain -> toneFilter
```

## Not changed

- No spectral filters were added.
- Spectral Engine faders remain UI-only.
- Mute/Unmute remains UI-only.
- The unstable Spectral Engine audio hook remains paused.
- No `AudioNode.prototype.connect` patch was added.
