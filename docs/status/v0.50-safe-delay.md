# v0.50 — Safe Delay

Scope: one safe effects feature.

Goal: make Delay work without adding feedback or changing the main audio graph file.

Implemented behaviour:

- Delay is now active.
- 0 means no audible delay.
- Higher values add a short, quiet echo.
- The dry path is unchanged.
- The delay wet gain is capped low.
- No feedback loop is connected.

Implementation:

- `src/delay-effect.js` installs a small runtime delay branch.
- The branch taps the existing post-fuzz output and feeds a quiet delayed signal into the existing master output.
- The existing Delay slider controls wet level and delay time.

Limits:

```text
maximum wet gain = 0.08
minimum delay time = 0.07 seconds
maximum delay time = 0.33 seconds
feedback = none
```

Preserved:

- no Reverb implementation
- no Band 5 audition
- no feedback loop
- no filter-bank behaviour
- no gain safety increase
- no deployment workflow changes

Test checklist:

- app loads
- oscillator starts and stops
- noise starts and stops
- Delay above 0 creates a short quiet echo
- Delay at 0 removes the echo
- Output still works
- Cutoff still works
- Virtual Distance still works
- Two-Moon Movement still works
- Panic Stop still works
- no console errors
