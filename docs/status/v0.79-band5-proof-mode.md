# v0.79 — Band 5 proof mode

Implements issue #130.

## Purpose

Manual test showed that the Band 5 test branch still did not produce an obvious audible change.

This build adds a temporary diagnostic proof mode so we can confirm whether Band 5 controls reach an audible node.

## Proof mode

Band 5 only:

```text
Band 5 fader -> temporary 1200 Hz proof tone gain
Band 5 mute  -> proof tone gain = 0
Panic Stop   -> proof tone gain = 0
```

## Important

This is diagnostic proof behaviour, not final Spectral Engine sound design.

## Scope kept

- Band 5 only.
- Bands 1–4 and 6–10 remain UI-only.
- No full ten-band engine.
- No vocoder.
- No global Web Audio connection hook.
- No core oscillator/noise/cutoff/resonance/fuzz routing changes.

## Required manual test

```text
Start Oscillator
Move Band 5 fader from 0 to 100
Confirm proof tone appears or changes clearly
Mute Band 5
Confirm proof tone disappears
Unmute Band 5
Confirm proof tone returns
Start Noise
Repeat Band 5 fader/mute test
Check Cutoff, Resonance, Buttery Fuzz still work
Check Panic Stop silences proof tone
```
