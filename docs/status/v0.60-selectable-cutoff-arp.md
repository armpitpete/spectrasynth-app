# v0.60 — Selectable Cutoff arpeggiator

Scope: Scale Chance / Cutoff movement only.

Implemented:

- Added Arp Mode Off / On.
- Added Arp Notes count from 2 to 12.
- Added Arp Note 1 to Arp Note 12 selectors.
- When Arp Mode is On, Scale Chance steps through the first selected N notes in order.
- Each selected note is converted to frequency and mapped to the existing Cutoff / Brightness slider.
- Note Length and Note Gap control timing.
- Rest Chance can skip a cycle.
- Repeat Chance can repeat the previous target.
- When Arp Mode is Off, the existing chance-based cutoff movement remains available.

Preserved:

- no oscillator pitch automation
- no new audio nodes
- no MIDI
- no sequencer grid
- no polyphony
- no presets
- no filter-bank behaviour
- no Band 5 audition
- no feedback loop
- no Delay, Reverb, or Stereo Width changes

Test:

- app loads
- Arp Mode appears
- Arp Notes count runs from 2 to 12
- Arp Note selectors are editable
- Scale Chance On plus Arp Mode On steps through selected notes
- Arp Mode Off keeps chance-based movement
- existing synth controls still work
