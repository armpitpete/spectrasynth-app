# v0.67 — High spectral note range

Scope: note selector range only.

Problem:

Scale Chance, Arp Notes, and Cluster Notes previously only offered notes up to B6. That did not reach far enough into the upper Cutoff / Brightness range to excite spectral bands 8, 9, and 10.

Implemented:

- Extended the shared note selector range from C1–B6 to C1–B9.
- Scale Chance Low Note / High Note can now select high notes up to B9.
- Arp Notes can now select high notes up to B9.
- Cluster Notes can now select high notes up to B9.
- Existing note-to-frequency-to-cutoff mapping is unchanged.

Preserved:

- no Scale Chance engine logic changes
- no arp behaviour changes
- no cluster behaviour changes
- no Cutoff mapping changes
- no oscillator pitch automation
- no new audio nodes
- no MIDI
- no sequencer grid
- no polyphony
- no presets
- no pattern saving
- no filter-bank behaviour
- no Band 5 audition
- no feedback loop
- no Delay, Reverb, or Stereo Width changes
- no deployment workflow changes

Test:

- app loads
- Low Note / High Note include notes up to B9
- Arp Note selectors include notes up to B9
- Cluster Note selectors include notes up to B9
- high selected notes move Cutoff / Brightness far enough to excite upper spectral bands
- existing Scale Chance / Arp / Cluster behaviour still works
