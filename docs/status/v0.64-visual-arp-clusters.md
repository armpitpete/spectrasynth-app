# v0.64 — Visual-only Arp Clusters panel

Scope: UI only.

Implemented:

- Added Arp Clusters section to Scale Chance.
- Added Cluster Mode Off / On.
- Added Cluster Count from 1 to 4.
- Added Cluster Size from 1 to 6.
- Added Cluster Direction: As Selected, Up, Down, Up / Down, Random.
- Added four clusters, each with six note selectors.
- Plain Patch Summary reports selected cluster settings.

Behaviour:

- Arp Clusters are visual-only in v0.64.
- They do not drive Cutoff / Brightness yet.
- They do not change existing Scale Chance arp behaviour yet.

Preserved:

- no Scale Chance engine changes
- no existing arp behaviour changes
- no Cutoff movement changes
- no oscillator pitch automation
- no new audio nodes
- no MIDI
- no sequencer grid
- no polyphony
- no presets
- no pattern saving
- no filter-bank behaviour
- no Band 5 audition
- no feedback loop
- no Delay, Reverb, or Stereo Width changes
- no deployment workflow changes

Test:

- app loads
- Arp Clusters section appears
- Cluster Mode can be changed
- Cluster Count can be set from 1 to 4
- Cluster Size can be set from 1 to 6
- Cluster Direction can be changed
- Cluster note selectors can be changed
- existing Scale Chance / Arp behaviour still works as before
