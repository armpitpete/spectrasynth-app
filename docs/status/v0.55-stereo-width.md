# v0.55 — Stereo Width

Scope: one safe stereo-output feature.

Goal: add a real Stereo Width control without changing the mono synth core.

Implemented behaviour:

- Added a visible Stereo Width slider in the Effects panel.
- 0 means narrow / mostly mono.
- 50 keeps the current normal stereo spread.
- 100 makes the existing stereo spread wider.
- The control changes only the existing centre gain, left/right spread gains, and left/right panners.

Preserved:

- no new audio nodes
- no Delay changes
- no Reverb changes
- no feedback loop
- no Band 5 audition
- no filter-bank behaviour
- no gain safety increase
- no deployment workflow changes

Test checklist:

- app loads
- Stereo Width slider appears above Output
- 0 sounds narrower / mostly mono
- 50 sounds like the previous normal spread
- 100 sounds wider
- Delay still works
- Reverb still works
- Panic Stop still works
- no console errors
