# v0.61 — Arp Direction modes

Scope: Scale Chance / Cutoff arpeggiator only.

Implemented:

- Added Arp Direction.
- Direction modes: As Selected, Up, Down, Up / Down, Down / Up, Random.
- As Selected follows the selected Arp Notes in panel order.
- Up sorts selected notes low to high.
- Down sorts selected notes high to low.
- Up / Down rises then falls without repeating the end notes twice inside the phrase.
- Down / Up falls then rises without repeating the end notes twice inside the phrase.
- Random chooses a selected arp note each cycle.

Preserved:

- no oscillator pitch automation
- no new audio nodes
- no MIDI
- no sequencer grid
- no polyphony
- no presets
- no filter-bank behaviour
- no Band 5 audition
- no feedback loop
- no Delay, Reverb, or Stereo Width changes

Test:

- app loads
- Arp Direction appears
- each direction changes the Cutoff arpeggio order
- Rest Chance and Repeat Chance still work
- Arp Mode Off still returns to chance-based cutoff movement
- existing synth controls still work
