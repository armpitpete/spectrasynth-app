# v0.76 — Unmute failure

Manual test after PR #120:

- Mute removes the band contribution.
- Unmute does not restore the band contribution.

Interpretation:

The remaining fault is state synchronisation between the visible mute button and the spectral audio gain update. The repair makes the mute visual layer dispatch a dedicated state-change event after the visible button state has updated, and makes the spectral audio layer update gains from that event.

Expected behaviour:

- Muting a band removes that band contribution.
- Unmuting the same band restores that band contribution if its fader is above zero.
- All-muted still silences the Spectral Engine contribution.

Scope remains v0.76 recovery only. Do not add new features.
