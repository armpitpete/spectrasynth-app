# v0.62 — Arp Notes 1 to 12 dropdown

Scope: small Arp Notes correction only.

Implemented:

- Arp Notes is now a drop-down menu.
- Arp Notes now allows 1 to 12 selected arp notes.
- The engine now accepts one selected arp note as valid.
- If only one arp note is selected, Arp Mode repeats that one Cutoff target rhythmically.

Preserved:

- no arp direction changes
- no note selector range changes
- no oscillator pitch automation
- no new audio nodes
- no MIDI
- no sequencer grid
- no polyphony
- no presets
- no filter-bank behaviour
- no Band 5 audition
- no feedback loop
- no Delay, Reverb, or Stereo Width changes

Test:

- app loads
- Arp Notes is a drop-down menu
- Arp Notes can be set to 1
- Arp Notes can be set to 12
- Arp Mode On works with one selected note
- existing arp direction modes still work
