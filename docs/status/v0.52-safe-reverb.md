# v0.52 — Safe Reverb

Scope: one safe effects feature.

Goal: make Reverb work without changing Delay behaviour or adding feedback.

Implemented behaviour:

- Reverb is now active.
- 0 means no audible reverb.
- Higher values add a small controlled room/space effect.
- The dry path is unchanged.
- The reverb wet gain is capped low.
- No feedback loop is connected.

Implementation:

- `src/reverb-effect.js` installs a small runtime reverb branch.
- The branch taps the existing post-fuzz output and feeds a quiet convolver room signal into the existing master output.
- The existing Reverb slider controls wet level.

Limits:

```text
maximum wet gain = 0.11
room impulse length = 1.15 seconds
pre-delay = 0.018 seconds
feedback = none
```

Preserved:

- no Delay behaviour changes
- no Band 5 audition
- no feedback loop
- no filter-bank behaviour
- no gain safety increase
- no deployment workflow changes

Test checklist:

- app loads
- oscillator starts and stops
- noise starts and stops
- Reverb above 0 creates a small controlled space effect
- Reverb at 0 removes the space effect
- Delay still works
- Output still works
- Cutoff still works
- Virtual Distance still works
- Two-Moon Movement still works
- Panic Stop still works
- no console errors
