# v0.42 — First instrument-panel visual polish

Scope: visual polish only.

Goal: make the public SpectraSynth page feel more like a compact instrument panel without changing sound behaviour.

Allowed changes:

- CSS surface polish
- tighter panel styling
- more hardware-like buttons, cards, and strips
- visible version checkpoint update

Not allowed:

- no audio code changes
- no Web Audio routing changes
- no gain changes
- no deployment changes
- no Band 5 audition
- no feedback
- no filter-bank behaviour

Good enough:

The app should still work exactly as before, but look less like a webpage mockup and more like an instrument panel.
