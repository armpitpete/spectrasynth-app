# v0.81 — Band 5 broad colour target

Implements issue #134.

Manual test after #132 / PR #133:

```text
still sounds like a fixed pitch
```

## Diagnosis

A source-fed fixed 1200 Hz bandpass can still sound like a fixed tone when fed by a pitched oscillator.

The repair makes Band 5 act more like broad colour/body by converting the Band 5 audition filter from a narrow bandpass into a wider peaking colour stage at runtime.

## Change

Band 5 only:

```text
1200 Hz peaking colour
Q 0.55
+5 dB peak gain
```

## Scope kept

- Band 5 only.
- Bands 1–4 and 6–10 remain UI-only.
- No full ten-band engine.
- No vocoder.
- No `AudioNode.prototype.connect` patch.
- No core oscillator/noise/cutoff/resonance/fuzz routing changes.

## Required manual test

```text

Confirm the sound changes as colour/body, not as a separate fixed pitch
```
