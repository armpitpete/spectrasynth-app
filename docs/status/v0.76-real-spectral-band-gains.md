# v0.76 — Real Spectral Engine band gains

Scope: first real spectral audio behaviour only.

Implemented:

- Added a gentle audible parallel spectral colour layer.
- Existing Spectral Engine faders now control each band contribution.
- Existing Mute/Unmute buttons now remove or restore each band contribution.
- The layer is created when oscillator/noise source gain connects into the main tone filter.
- The layer feeds back into the existing tone/filter/fuzz/effects path, so existing Cutoff, Buttery Fuzz, Delay, Reverb, Stereo Width, Output, and safety behaviour remain in charge.
- Band gains are intentionally low and shaped for safety.
- Spectral Engine wording now says the faders affect sound.

Band centres:

- Band 1 Low: 80 Hz
- Band 2 Low-Mid: 160 Hz
- Band 3 Body: 320 Hz
- Band 4 Warmth: 640 Hz
- Band 5 Voice: 1200 Hz
- Band 6 Edge: 2200 Hz
- Band 7 Presence: 3800 Hz
- Band 8 Bright: 6200 Hz
- Band 9 Air: 9000 Hz
- Band 10 Hiss: 12500 Hz

Preserved:

- no oscillator pitch automation
- no Scale Chance behaviour changes
- no AR envelope changes
- no arp or cluster changes
- no Source Readout changes
- no Delay changes
- no Reverb changes
- no Stereo Width changes
- no MIDI
- no presets
- no deployment workflow changes

Limit:

This is not a full vocoder or full replacement all-band filter bank. It is the first safe audible spectral colour layer.

Test:

- app loads
- Start Oscillator works
- Start Noise works
- moving Spectral Engine faders changes tone audibly
- muting a band removes that band contribution
- unmuting restores it
- Delay/Reverb/Stereo Width still work
- Scale Chance and AR Envelope still work
