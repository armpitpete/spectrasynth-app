# v0.78 — Band 5 spectral test branch

Implements issue #127.

## Change

Adds one safe audible Spectral Engine test band only:

```text
sourceMixGain -> spectralBand5Filter -> spectralBand5Gain -> existing stereo/output path
```

Band 5 uses:

```text
frequency = 1200 Hz
Q = 1.2
max gain = 0.1
```

## Fader and mute behaviour

Only Band 5 affects audio.

```text
Band 5 fader -> spectralBand5Gain
Band 5 mute  -> spectralBand5Gain = 0
```

Bands 1–4 and 6–10 remain UI-only.

## Protected core path

The core path remains:

```text
sourceMixGain -> toneFilter -> buttery fuzz -> post-filter -> stereo/output
```

## Not changed

- No ten-band engine.
- No vocoder.
- No global Web Audio connection hook.
- No preset changes.
- No all-band fader audio.
