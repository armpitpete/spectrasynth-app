# v0.59 — Scale Chance rhythmic Cutoff movement

Scope: first real Scale Chance behaviour.

Decision: Scale Chance controls Cutoff / Brightness first, not oscillator pitch.

Implemented:

- Scale Chance On / Off now controls a rhythmic Cutoff movement engine.
- Root and Scale define the allowed musical targets.
- Low Note and High Note constrain the target range.
- Chosen notes are converted to frequency, then mapped to the existing Cutoff / Brightness slider.
- Existing cutoff input events are used, so the current safe filter logic remains in charge.
- Randomness affects jump size.
- Pitch Centre biases the target area.
- Note Length and Note Gap control timing.
- Rest Chance can skip a movement cycle.
- Repeat Chance can repeat the previous target.

Preserved:

- no oscillator pitch automation
- no new audio nodes
- no MIDI
- no sequencer grid
- no polyphony
- no presets
- no filter-bank behaviour
- no Band 5 audition
- no feedback loop
- no Delay, Reverb, or Stereo Width changes

Test:

- app loads
- Scale Chance panel shows Rest Chance and Repeat Chance
- Scale Chance Off leaves manual Cutoff working
- Scale Chance On moves Cutoff rhythmically
- other existing controls still work
