# v0.75 — Spectral Engine mute/unmute controls

Scope: Spectral Engine mute control clarity only.

Implemented:

- Added Spectral Engine mute control helper module.
- Muted bands now show an **Unmute** button label.
- Active bands show a **Mute** button label.
- Muted band strips are visually dimmed.
- Muted band meter fills are visually suppressed.
- `aria-pressed` remains controlled by the existing main Spectral Engine state.
- Spectral Engine note wording now explains this is visible band mute state.

Preserved:

- no audio filter-bank behaviour
- no oscillator pitch changes
- no Scale Chance changes
- no AR envelope changes
- no arp or cluster changes
- no Source Readout changes
- no Delay changes
- no Reverb changes
- no Stereo Width changes
- no deployment workflow changes

Note:

This is not full audio band muting. Full audio band mute needs the later all-band filter-bank engine.

Test:

- app loads
- each Spectral Engine band starts with Mute
- clicking Mute changes the button to Unmute
- muted band strip dims
- muted meter fill is visually suppressed
- clicking Unmute returns the button to Mute
- existing audio and controls still work
