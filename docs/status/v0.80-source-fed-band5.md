# v0.80 — Source-fed Band 5 audition branch

Implements issue #132.

## Purpose

Replace the temporary fixed-pitch Band 5 proof tone with the source-fed Band 5 audition branch.

Manual test after #130 / PR #131 correctly reported:

```text
It's just a fixed pitch
```

That fixed pitch was diagnostic only. It proved control wiring but was not real Spectral Engine behaviour.

## Change

Removed the separate fixed 1200 Hz proof oscillator from `src/spectral-band-audio.js`.

Band 5 now relies on the source-fed path already built in `src/main.js`:

```text
sourceMixGain -> spectralBand5Filter -> spectralBand5Gain -> existing stereo/output path
```

## Expected behaviour

Band 5 should now process the actual source:

- Oscillator when Oscillator is running
- Noise when Noise is running
- both if both are running

It should not produce an independent fixed sine tone.

## Scope kept

- Band 5 only.
- Bands 1–4 and 6–10 remain UI-only.
- No full ten-band engine.
- No vocoder.
- No global Web Audio connection hook.
- No core oscillator/noise/cutoff/resonance/fuzz routing changes.

## Required manual test

```text
Start Oscillator
Move Band 5 fader from 0 to 100
Confirm the sound changes as processed oscillator colour, not a separate fixed sine pitch
Mute Band 5
Confirm the Band 5 contribution disappears
Unmute Band 5
Confirm it returns
Start Noise
Repeat Band 5 fader/mute test
Check Cutoff, Resonance, Buttery Fuzz still work
Check Panic Stop
```
