# v0.68 — Scale Chance AR envelope trigger

Scope: Scale Chance note-event articulation only.

Implemented:

- Added AR Envelope controls to the Scale Chance panel.
- Added AR Envelope Off / On.
- Added Attack control.
- Added Release control.
- Scale Chance now dispatches a note event when it chooses a real musical target.
- Rest cycles do not dispatch note events.
- A new AR envelope module listens for Scale Chance note events.
- When AR Envelope is On, each note event shapes Output with attack and release.
- Panic Stop cancels the envelope and drops Output to zero.
- Turning AR Envelope Off restores Output to its stored base value.

Behaviour:

- Chance notes trigger the AR envelope.
- Arp notes trigger the AR envelope.
- Cluster notes trigger the AR envelope.
- Rest Chance does not trigger the envelope.
- The envelope uses Output as a safe articulation stage and does not add new audio nodes.

Preserved:

- no oscillator pitch automation
- no new audio nodes
- no MIDI
- no sequencer grid
- no polyphony
- no presets
- no pattern saving
- no filter-bank behaviour
- no Band 5 audition
- no feedback loop
- no Delay, Reverb, or Stereo Width changes
- no deployment workflow changes

Test:

- app loads
- AR Envelope controls appear
- AR Envelope Off preserves existing Scale Chance behaviour
- AR Envelope On shapes generated notes with attack and release
- Attack changes how fast each event opens
- Release changes how fast each event falls away
- Rest Chance does not trigger the envelope
- Panic Stop cancels/silences the envelope
- existing Scale Chance / Arp / Cluster behaviour still works
