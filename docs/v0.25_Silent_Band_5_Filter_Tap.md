# v0.25 Silent Band 5 Filter Tap

This note records the v0.25 spectral-engine wiring proof.

## Goal

Add the first real spectral-band filter node without changing the audible sound.

## What changed

Band 5 — Voice now has a real internal silent filter tap.

```text
Band 5 filter type = bandpass
Band 5 centre frequency = 1200 Hz
Band 5 Q = 1.2
```

The Band 5 tap is fed from the post-fuzz signal, before the stereo/output branches.

Its output is routed only to an internal zero-gain path.

It is not connected to `masterGain`.

## Preserved audible path

```text
oscillator / noise → pre-fuzz low-pass filter → soft Buttery Fuzz mix → post-fuzz low-pass filter → true left/right stereo spread → master Output → analyser / speakers
```

## Important rule

The Band 5 filter tap is not audible.

Band 5 fader and Band 5 Mute still do not control sound.

This is only a wiring proof for future spectral-band work.
