# v0.22 Stable Cutoff and Buttery Fuzz Checkpoint

This note records the v0.22 checkpoint for SpectraSynth.

## Status

The tested cutoff-response and Buttery Fuzz state is frozen.

No sound behaviour changes are intended in v0.22.

## Preserved audio path

```text
oscillator / noise → pre-fuzz low-pass filter → soft Buttery Fuzz mix → post-fuzz low-pass filter → true left/right stereo spread → master Output → analyser / speakers
```

## Preserved cutoff behaviour

Cutoff / Brightness uses a 0–100 slider mapped perceptually from 120 Hz to 16000 Hz.

The default slider value is 63, which keeps the starting tone near the earlier 2600 Hz position.
