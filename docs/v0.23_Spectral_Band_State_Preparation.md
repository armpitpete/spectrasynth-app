# v0.23 Spectral Band State Preparation

This note records the v0.23 preparation step for SpectraSynth.

## Goal

Prepare the 10-band spectral panel for future real band behaviour without changing the sound engine.

## Status

The spectral panel is now stateful.

The 10 meters still respond to real analyser data.

The 10 faders and Mute buttons remain visual-only.

## What each band stores

Each spectral band stores:

```text
band number
label
fader value
muted state
analyser level
```

## What faders do now

Moving a band fader updates the stored visual fader value and the plain patch summary.

It does not change the audio.

## What Mute buttons do now

Clicking a Mute button toggles the stored muted state, changes the button text/appearance, and updates the plain patch summary.

It does not mute audio.

## Preserved audio path

```text
oscillator / noise → pre-fuzz low-pass filter → soft Buttery Fuzz mix → post-fuzz low-pass filter → true left/right stereo spread → master Output → analyser / speakers
```

## Still not built

- real 10-band filters
- band fader audio behaviour
- band mute audio behaviour
- vocoder
- microphone input
- MIDI
- presets
- sensors
- feedback reconnection
