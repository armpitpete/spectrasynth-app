# v0.21 Cutoff Response Curve

## Goal

Make Cutoff / Brightness usable across the full slider.

## Problem found in testing

The previous Cutoff slider reached its useful peak around 50%.

That happened because the slider used a direct linear Hz range from 120 Hz to 16000 Hz. Halfway up the slider was already around 8000 Hz, so the top half did not feel very useful.

## Change

Cutoff / Brightness now uses a 0 to 100 slider and maps that value to 120 Hz to 16000 Hz with a perceptual curve.

The default slider value is 63, which keeps the starting tone close to the old 2600 Hz default.

## Preserved

This change does not retune Buttery Fuzz, Resonance, stereo spread, Output, Panic Stop, analyser meters, or the parked feedback state.

## Test checklist

- Cutoff changes the sound across the full slider travel.
- The useful peak is no longer reached at 50%.
- Low settings still sound dark.
- High settings still sound bright.
- Buttery Fuzz still sounds like the tested v0.20 version.
- Resonance still works.
- Output works.
- Panic Stop works.
- Feedback remains parked.
